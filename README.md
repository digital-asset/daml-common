# Daml Common

Common libraries for Daml products. Run `build` to build.

## Setting up this repo locally

### Prerequisites

* Install [direnv](https://github.com/direnv/direnv/blob/master/docs/installation.md).
* Install the [Nix](https://nixos.org/download.html) package manager.

`direnv` and `Nix` will work together to provide you with an appropriate,
working environment.
